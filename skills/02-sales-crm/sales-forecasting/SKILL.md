# Sales Forecasting Skill

## Forecast Methods
- Top-down: Target × % = Forecast
- Bottom-up: Deal by deal
- Run-rate: Current + new
- Pipeline: Weighted value

## Weighted Pipeline
Closed = % × Deal Value
100% = Verbal commitment
75% = Verbal discussion
50% = Strong fit + timeline
25% = Qualified

## Forecasting Best Practices
- Weekly forecast calls
- Deal-by-deal review
- Commit vs Best vs Pipeline
- Always track accuracy
