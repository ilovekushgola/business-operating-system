# Pipeline Management Skill

## Pipeline Stages
1. Discovery: Fit confirmed
2. Qualification: BANT done
3. Proposal: Solution presented
4. Negotiation: Terms discussed
5. Closed Won: Contract signed
6. Closed Lost: No deal

## Pipeline Metrics
- Deals by stage
- Avg deal size
- Cycle length
- Win rate
- Activity metrics

## Deal Review Questions
- What's the close plan?
- What's the competition?
- What's the risk?
- What's the commitment?

## Forecast Logic
- Commit: Verbal + timeline
- Best case: Qualified + verbal
- Pipeline: All deals
