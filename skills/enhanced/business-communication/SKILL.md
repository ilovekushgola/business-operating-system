# Business Communication Skill

## Writing Principles

1. Clear - No jargon
2. Concise - Get to point
3. Action-oriented - What's needed

## Document Types

| Type | Purpose | Length |
|------|--------|--------|
| Email | Quick comms | 3-5 sentences |
| Memo | Decision needed | 1 page |
| Proposal | Approval | 2-5 pages |
| Deck | Presentation | 10-15 slides |

## Presentation Framework

1. Hook (attention)
2. Problem (why it matters)
3. Solution (what we do)
4. Proof (evidence)
5. Ask (call to action)
