# Data Visualization Skill

## Chart Selection

| Data Type | Best Chart |
|----------|-----------|
| Trend | Line |
| Comparison | Bar |
| Proportion | Pie/Donut |
| Distribution | Histogram |
| Correlation | Scatter |

## Dashboard Principles

1. One story per dashboard
2. Most important at top
3. Context + numbers
4. Actionable insights

## Tools

- Tableau, PowerBI: Enterprise
- Metabase: Open source
- Grafana: Metrics
