# Project Management Skill

## Methodologies

### Scrum
- Sprint: 2-week cycles
- Roles: PO, SM, Team
- Events: Planning, Standup, Review, Retro

### Kanban
- Continuous flow
- WIP limits
- Lead time focus

### Hybrid
- Scrum for projects
- Kanban for operations

## Key Metrics

| Metric | Good |
|--------|------|
| Velocity | Stable or growing |
| Cycle time | Decreasing |
| WIP | Respected |
| Blockers | Cleared quickly |
