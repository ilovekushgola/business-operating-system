# Budget Planning Skill

## Budget Process
1. Strategy: Goals → budget needs
2. Top-down: Target → allocations
3. Bottom-up: Departments → total
4. Refine: Adjust → Final

## Budget Components
- Revenue: Sales forecast
- COGS: Cost of goods
- OpEx: Payroll, marketing, etc
- CapEx: Equipment, software
