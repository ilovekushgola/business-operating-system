# CFO Expert Skill

Comprehensive CFO expertise covering financial strategy, forecasting, investor relations, and capital management.

## When to Use This Skill

USE THIS SKILL when:
- Financial strategy and planning
- Investor relations and fundraising
- Financial forecasting and modeling
- Board financial presentations
- M&A due diligence
- Capital allocation decisions
- "What's our financial strategy?"
- "How does this deal look financially?"
- Year-end financial planning

DO NOT USE for:
- Day-to-day bookkeeping (use bookkeeper)
- Tax preparation (use tax-professional)
- Payroll (use payroll-suite)

## Financial Strategy

### Strategic Financial Framework

```
1. Assess Current State → 2. Define Targets → 3. Develop Strategy → 4. Execute → 5. Measure
```

### Step 1: Current State Assessment

**Financial Health Checklist:**

| Area | Metrics | Status |
|------|--------|--------|
| Liquidity | Cash, runway | [X months] |
| Profitability | Gross margin, EBITDA | [X]% |
| Efficiency | CAC, LTV, burn | [X] |
| Growth | Revenue growth, MRR | [X]% |
| Capital | Debt, equity | $[X] |

**SWOT Analysis:**
```markdown
## Financial SWOT

### Strengths
- [Cash position/reserves]
- [Revenue diversification]
- [Unit economics]

```
