# Financial Modeling Skill

## Model Types
- P&L: Revenue → Profit
- Cash flow: Operating, investing, financing
- Balance sheet: Assets, liabilities, equity
- Cap table: Equity splits

## Model Components
- Assumptions: Revenue, costs
- Scenarios: Base, best, worst
- Drivers: Growth rate, CAC, etc
- Outputs: Key metrics
