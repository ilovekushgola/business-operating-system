# Vendor Management Skill

## Vendor Lifecycle
- Selection: RFP, evaluation
- Onboarding: Contract, setup
- Management: Performance
- Renewal: Review, renegotiate

## Evaluation Criteria
- Price: Cost vs value
- Quality: SLA adherence
- Reliability: Uptime, support
- Fit: Integration
