# Project Planning Skill

## Project Phases
1. Initiation: Scope, stakeholders
2. Planning: Tasks, timeline
3. Execution: Do the work
4. Monitoring: Track status
5. Closing: Review, docs

## Planning Templates
- Charter: Scope, goals
- WBS: Task breakdown
- Timeline: Milestones
- RAID: Risks, assumptions
