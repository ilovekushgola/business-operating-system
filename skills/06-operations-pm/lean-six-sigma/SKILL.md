# Lean Six Sigma Skill

## Lean Principles
1. Value: What matters to customer
2. Value stream: Steps that add value
3. Flow: Keep moving
4. Pull: Just in time
5. Perfect: Continuous improvement

## Six Sigma (DMAIC)
- Define: Problem
- Measure: Current state
- Analyze: Root cause
- Improve: Fix
- Control: Sustain
