## Copywriting Frameworks

### AIDA

```
A - Attention (Headline)
I - Interest (Hook)
D - Desire (Benefits)
A - Action (CTA)
```

### PAS

```
P - Problem
A - Agitation
S - Solution
```

### Feature-Benefit-Value

```
Feature: [What it is]
Benefit: [What it does for user]
Value: [Why it matters]
```

## SEO Best Practices

### Keyword Research

| Type | Example | Intent |
|------|---------|--------|
| Short-tail | "CRM" | Informational |
| Mid-tail | "CRM for startups" | Informational |
| Long-tail | "best CRM for SaaS startups" | Commercial |

### On-Page SEO Checklist

- [ ] Title tag (60 chars)
- [ ] Meta description (155 chars)
- [ ] H1 (only one)
- [ ] H2s (logical structure)
- [ ] Keyword in first 100 words
- [ ] Keyword in H2/H3
- [ ] Images with alt text
- [ ] Internal links (2-4)
- [ ] External links to authority
- [ ] Readable URL slug
