# Troubleshooting Skill

## Troubleshooting Steps
1. Identify: What's the issue?
2. Isolate: Reproduce the problem
3. Analyze: Find root cause
4. Fix: Solve
5. Verify: Confirmed fixed

## Common Issues
- Login: Password, 2FA, browser
- Performance: Cache, network
- Payments: Card, processing
- Integration: API, auth
