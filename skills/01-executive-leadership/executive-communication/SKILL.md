# Executive Communication Skill

## Communication Types
- All-hands: Weekly/monthly
- Board: Monthly
- Town hall: Quarterly
- Press: As needed
- LinkedIn: Ongoing

## Presentation Framework
```
1. Hook (attention)
2. Problem (why it matters)
3. Solution (what we do)
4. Proof (evidence)
5. Ask (call to action)
```

## Speaking Tips
- Start with impact
- Simple, clear
- End with action
- Q&A: Bridge to key messages
