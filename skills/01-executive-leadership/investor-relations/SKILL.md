# Investor Relations Skill

## IR Cadence
- Monthly: Quick updates
- Quarterly: Board deck
- Annually: All-hands review

## Investor Types
- Angels: Hands-on
- VCs: Quarterly
- PE: Monthly
- Public: Quarterly (SEC)

## Update Template
```
Highlights
- Wins this month
Metrics
- Revenue, growth, customers
Challenges
- What we learned
Ask
- How investors can help
```

## Key Metrics
- ARR, MRR
- Net revenue retention
- Customer count
- Burn rate, runway
