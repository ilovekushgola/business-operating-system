# Crisis Management Skill

## Crisis Levels
- L1: TeamIncident - Team handles
- L2: Department - Manager
- L3: Company - Executive
- L4: Board/Public - Full crisis

## Crisis Response
```
1. Assess → 2. Contain → 3. Resolve → 4. Communicate → 5. Learn
```

## Communication Template
- Internal: Quick, clear, action-oriented
- External: Prepared, consistent, factual
- PR needed: Direct to GC/PR team

## Common Crises
- Security breach
- Key person loss
- Financial issue
- PR issue
- Legal issue
