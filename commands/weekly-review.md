# /weekly-review - Weekly Performance Review

End of week team retrospective.

## Usage

```bash
/weekly-review        # Start review
/weekly-review report # Complete report
/weekly-review team   # By team
```

## Review Categories

### Metrics
- Goals vs actual
- Key performance indicators
- Progress on OKRs

### Wins
- Celebrate successes

### Learnings
- What worked
- What didn't

### Action Items
- Next week priorities
