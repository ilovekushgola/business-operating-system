# /marketing-campaign - Marketing Campaign Manager

Create and launch marketing campaigns.

## Usage

```bash
/marketing-campaign create    # New campaign
/marketing-campaign status   # View campaigns
/marketing-campaign pause    # Pause campaign
/marketing-campaign resume   # Resume campaign
```

## Campaign Types

- Email: Email sequences
- Social: Social media
- Ads: Paid advertising
- Content: Content marketing
- SEO: Search optimization

## Create Campaign Template

```
Name: [Campaign Name]
Type: [Email/Social/Ads/Content/SEO]
Audience: [Target segment]
Duration: [Start - End]
Budget: [Amount]
```
