# /onboard - Employee Onboarding

Onboard new employees with proper setup.

## Usage

```bash
/onboard create    # Start onboarding
/onboard checklist # View tasks
/onboard status   # Progress
/onboard complete # Mark done
```

## Onboarding Checklist

### Day 1
- [ ] Paperwork completed
- [ ] Laptop and equipment
- [ ] Email setup
- [ ] Slack/Teams access

### Week 1
- [ ] Team introduction
- [ ] Tool access
- [ ] Training completed
- [ ] Manager 1:1

### First 30 Days
- [ ] First project assigned
- [ ] 30-day goals set
- [ ] Performance check-in
