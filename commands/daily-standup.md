# /daily-standup - Daily Team Sync

Start the day with team sync.

## Usage

```bash
/daily-standup        # Start standup
/daily-standup report # Generate report
/daily-standup team   # Team members
```

## Standup Format

### Yesterday
- What was completed

### Today
- What's planned

### Blockers
- Anything blocking progress

## Team Categories

- Sales: Deals, pipeline
- Marketing: Campaigns, leads
- Engineering: Features, bugs
- Operations: Tasks, projects
